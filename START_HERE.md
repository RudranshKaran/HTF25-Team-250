## ⚡ Quick Start (5 minutes)

### Step 1: Install Dependencies
```powershell
cd backend
pip install -r requirements.txt
```

### Step 2: Start Backend
```powershell
python main.py
```

### Step 3: Start Frontend (new terminal)
```powershell
cd frontend
npm run dev
```

### Step 4: Open Browser
```
http://localhost:5173
```

### Step 5: Click AI Insights
```
Sidebar → 🤖 AI Insights
Or press: 6
```

## 🔄 How It Works

```
1. You use dashboard
   ↓
2. Click "AI Insights" tab
   ↓
3. Frontend sends data to backend
   ↓
4. Backend calls Gemini AI (or fallback)
   ↓
5. AI analyzes crowd patterns
   ↓
6. Returns smart recommendations
   ↓
7. You see beautiful results
   ↓
8. You act on insights
   ↓
9. Crowd is managed better! ✅
```
